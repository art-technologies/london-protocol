# Verse NFT contracts

Verse ERC721 factory and protocol.

```shell
npx hardhat help
npx hardhat test
REPORT_GAS=true npx hardhat test
npx hardhat node
npx hardhat run scripts/deploy.ts
```

```shell
npx hardhat compile
```

```shell
npx hardhat test ./test/LondonTokenFactory.ts
```
